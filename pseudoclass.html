Catatan Penggunan Pseudo Class pada tag a

  • a:hover => harus meletakkan kursor pada link dan tidak perlu di klik
  • a:link => warna default dari link sebelum di klik atau di kunjungi
  • a:visited => warna link akan berubah setelah di klik atau dikunjungi
  • a:active => warna link akan berubah jika di klik kanan
